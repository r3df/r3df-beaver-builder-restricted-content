Role based content restrictions for Beaver Builder, with content teaser.


== Description ==
Adds a custom taxonomy to allow tagging of content for role based access restrictions.  Custom content
teasers can be created for Beaver Builder pages.  They will be shown instead of the full content when
the page is blocked.


== Installation ==
= The easy way: =
1. To install this plugin, click on "Add New" on the plugins page in your WordPress dashboard.
1. Search for "R3DF Beaver Builder", click install when it's found.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Configure the plugin in the settings page.

= The hard way: =
1. Download the latest r3df-beaver-builder-restricted-content.zip from wordpress.org
1. Upload the zip file to the `/wp-content/plugins/` folder on your web server
1. Uncompress the zip file and delete it after it's uncompressed
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin in the settings page.
